# 📊 HR Analytics Dashboard (Power BI)

## 📌 Overview
This project focuses on building an interactive HR Analytics dashboard to track key workforce metrics and support data-driven decision making.

---

## 🎯 Problem Statement
The HR department lacked clear performance indicators to monitor workforce health, attrition, and employee demographics.

This project defines and visualizes key KPIs to improve workforce planning and retention strategies.

---

## 📈 Key KPIs
- Employee Count
- Attrition Count
- Attrition Rate
- Active Employees
- Average Age

---

## 📊 Dashboard Features
- Attrition by Gender
- Department-wise Attrition
- Employees by Age Group
- Job Satisfaction Ratings
- Education Field-wise Attrition
- Attrition Rate by Gender & Age Group

---

## 🧠 Insights
- Identified departments with high attrition
- Highlighted age-group trends in employee turnover
- Provided gender-based attrition comparison
- Improved visibility into employee satisfaction

---

- Power BI
- Data Modeling
- Data Visualization
- KPI Design
